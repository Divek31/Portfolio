# Divek Kumar | Portfolio

Welcome to my personal, ultra-premium portfolio website! 👋 

This site is meticulously crafted to showcase who I am, the AI-powered applications I build, and my extensive skills in Python, Machine Learning, and Software Engineering. The design heavily leans into an awwwards-winning "Corn Revolution" aesthetic—featuring dynamic typography, immersive physics-based scrolling, and complex custom cursor interactions, all tied together by a soothing Mid-Century coastal color palette (Charcoal, Jasmine, Burnt Peach).

## 🌐 Live Demo
Check it out here: [https://divek31.github.io/Portfolio/](https://divek31.github.io/Portfolio/)

## 🚀 Built With
- **HTML5 & Vanilla CSS3:** Utilizing custom CSS properties for theme generation alongside complex `@keyframes`.
- **Vanilla JavaScript:** Building out interactive logic avoiding heavy framework bloatedness.
- **GSAP (GreenSock):** Handling all timeline loading animations, scroll triggers, text reveals, and DOM morphing.
- **Lenis:** Momentum-based, physics-driven smooth scroll engine to mimic an ultra-smooth native application experience.

## 📌 Features
- **Magnetic SVGs & Custom Cursor:** The cursor snaps to navigation links, buttons, and GitHub icons. It includes an SVG follower that acts as a scroll-progress monitor!
- **Fluid Lenis Scrolling:** Disables jerky native scrolling in favor of butter-smooth physics momentum.
- **Holographic 3D Cards:** The experience and certification cards leverage GSAP rotation trackers allowing the cards to tilt inside 3D space with a dynamic glossy glare mapping your mouse position!
- **Dynamic CSS Marquee:** A customized continuous sliding ticker providing depth under the hero banner context (`AVAILABLE FOR WORK • AI DEVELOPMENT • SOFTWARE ENGINEEING`).
- **Advanced Navigation Flips:** High-end typography animation where hovered links roll text dynamically away to replace them from the bottom up!
